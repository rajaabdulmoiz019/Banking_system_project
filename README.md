# Banking_system_project
This project demonstrates a basic Bank Account system implemented using Object-Oriented Programming (OOP) concepts in Python.<br><br>

The program allows a user to:<br>
• Deposit money into an account<br>
• Withdraw money from an account<br>
• Check and update the account balance securely using an account number<br><br>

**Key Features**<br>

• Uses a Python class (Account) to model a real-world bank account<br>
• Implements __init__ constructor for account setup<br>
• Supports deposit and withdraw operations<br>
• Validates account number before performing transactions<br>
• Prevents withdrawal if balance is insufficient<br><br>

 **Concepts Used**<br>

• Classes and Objects<br>
• Instance Variables<br>
• Methods<br>
• Conditional Statements (if / else)<br>
• Basic input validation<br><br>
